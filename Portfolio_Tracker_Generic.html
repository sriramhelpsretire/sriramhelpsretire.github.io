Portfolio Tracker

🏠 HOME ⬇ Download CSV Template
📂 Getting Started with Your Portfolio
1.Click the Download CSV Template button at the top right to get a sample structured file. 2.Open it in Excel, Google Sheets, or a text editor to input your personal holdings, keeping the column headers exactly as they are. 3.Click Import CSV File in the control toolbar below to instantly populate your tracker dashboard. 4.Alternatively, dynamically add or edit lines using the on-screen form, then use Export CSV to save your work for future sessions.
🔄 Resetting or Reloading Data
1.Click Clear All to wipe all current session data and start with an empty board. 2.Click Import CSV File and open your custom saved *.csv asset file to completely load your previous setup.
🔑 Setting Up Live Price Refresh (API Key)
If prices are not refreshing, you need a free API key from Finnhub:
1.Go to finnhub.io 2.Click "Get free API key" and sign up. 3.Paste the key into the API Key box on this page and click Save API Key.
🔍 Using Portfolio Query
Use Portfolio Query to filter and search your investments by Ticker, Account, or Asset Type. Click Run Query to see all matching holdings — great for answering quick questions about your portfolio.
💵 Pro Tip — Calculator Integration
Keep this Portfolio Tracker open while using the Retirement Calculator or Retirement Confidence calculator — key values from your portfolio will be pre-filled automatically, saving you time.
